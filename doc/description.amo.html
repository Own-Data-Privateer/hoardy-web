pwebarc (Private Web Archive): a suite of tools to collect, archive, and manage your web browsing data. Basically, it is your own private Wayback Machine that can also archive POST requests, or HTTP-level WebScrapBook following "archive everything now, figure out what to do with it later" philosophy. This extension only collects dumps of HTTP requests and responses as you browse the web. To archive, manage, and browse the archived data you'll need to use accompanying archiving server and related tools. To quickstart collecting your data, you need to install this extension and run at least the the dumb archiving server, which is a tiny Python script that uses the Python's standard library and nothing else. Have Python installed? You can run it right now, no need to install anything else. More info at https://github.com/Own-Data-Privateer/pwebarc.
